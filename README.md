# Maze Generator using python:

This is a Maze generator and solution finder made using python. This uses Randomized depth-first search Algorithm and Recursive implementation.

![maze-generator](maze_generator.gif)

## Installation:

You can try this yourself by any of these methods:
- clone this repo and run the code
- download the latest release and run the .exe
